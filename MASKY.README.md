# For‑Loop Counter
This program uses a for‑loop to count from 1 up to 10 by using the range(1, 11) function.

## Features
- Uses a for loop
- Counts from 1 to 10
- Ignores the number 11 because range(1, 11) stops at 10
- Prints each loop iteration


## Learning Purpose
- A small exercise to practice:
- for‑loops
- the range() function
- understanding loop boundaries
